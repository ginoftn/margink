# Margink

A markdown editor where your AI assistant understands your project, not just your file.

**Status:** In development. Nothing to install yet.

Website: [margink.com](https://margink.com)

---

Margink is a macOS markdown editor with a contextual AI assistant. The assistant automatically detects your project structure, style, and previous work — so you never have to re-explain context.

- Model-agnostic: Apple Intelligence, Ollama, OpenAI, Anthropic
- Privacy-first: on-device by default
- Free on the Mac App Store (when ready)
- Open source

More details on the website.
